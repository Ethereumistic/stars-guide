# Stars.Guide: Compositional Schema Definition

## 1. The Core Architecture
Astrology interpretations in the stars.guide engine are not pre-written paragraphs. They are dynamic equations built from atomic data blocks. 

The formula is always:
**[PLANET: The Drive] + [SIGN: The Style] + [HOUSE: The Arena] = [SYNTHESIS: The Output]**

To ensure the engine can build grammatically correct and psychologically profound sentences without human intervention, all AI-generated content MUST strictly adhere to the structural and grammatical schemas below.

---

## 2. The Grammatical Connectors
For the engine to construct sentences on the fly, every entity must provide a specific grammatical fragment.
* **Planets** provide a *Compositional Verb Phrase* (e.g., "seeks validation by", "processes emotional security through").
* **Signs** provide a *Compositional Adverbial Phrase* (e.g., "with relentless momentum", "through methodical stabilization").
* **Houses** provide a *Compositional Prepositional Phrase* (e.g., "within the realm of public status", "in the foundation of the home").

**Engine Synthesis Example:**
`Your core identity (Sun) [seeks validation by] [initiating with relentless momentum], particularly manifesting [within the realm of public status and career].`

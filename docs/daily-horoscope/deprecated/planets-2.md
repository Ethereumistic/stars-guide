# The 10 Celestial Bodies of Astrology

> Note: The Sun and Moon are not planets astronomically, but are treated as such in astrology. Pluto is a dwarf planet. All 10 are standard in astrological practice.

---

## 1. ☉ The Sun

### Astronomy
| Property | Value |
|---|---|
| Type | G2V Yellow Dwarf Star |
| Distance from Earth | 1 AU / ~149.6 million km |
| Diameter | 1,392,700 km (109× Earth) |
| Mass | 1.989 × 10³⁰ kg |
| Surface Temperature | 5,778 K |
| Core Temperature | ~15.7 million K |
| Age | ~4.603 billion years |
| Apparent Magnitude | −26.74 |
| Zodiac cycle | ~365.25 days (1 year per full circuit) |

### Astrology
| Property | Value |
|---|---|
| Archetype | The Self / The King |
| Rules | Leo |
| Element | Fire |
| Controls | Identity, ego, willpower, life force, self-expression |
| Body | Heart, spine, circulatory system |
| In birth chart | Core personality, conscious self, purpose |
| Polarity | Masculine / Yang |

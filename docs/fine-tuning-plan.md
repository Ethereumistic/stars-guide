Settings Page UI/UX Specification

This document outlines the redesign for the /settings page, focusing on a modern, card-based interface with responsive navigation.

1. Navigation Architecture

Desktop Layout (Vertical Tabs)

Position: Fixed left sidebar or left-aligned column.

Style: steal the design and style from the provided pages as context.

Naming (Required):

You (User details & Security)

Charts (Birth chart data)

Plan (Billing/Subscription)

Friends (Friends & Family keep that empty for now.)

Help (Support/FAQ)

Mobile Layout (Footer Ribbon)

Position: Sticky bottom navigation bar.

Behavior: Icons with short labels visible at all times.

Haptics: Provide visual feedback (active tint) on the current icon.
